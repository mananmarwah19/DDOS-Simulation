# DDoS Attack Simulation and Analysis

This project demonstrates simulated DDoS attack patterns using Python and analyzes the generated network traffic using Wireshark.

## Attacks Simulated
1. HTTP Flood Simulation
2. TCP SYN Traffic Simulation

## Tools Used
- Python
- Wireshark
- GitHub

## Objective
To understand how abnormal traffic patterns appear in packet capture tools and how they can be analyzed for network security purposes.
